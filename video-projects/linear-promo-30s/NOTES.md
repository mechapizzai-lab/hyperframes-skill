# linear-promo-30s — Notes

`final.mp4` in this project is a **draft-quality render** (CRF 28, ~3 MB). No standard-quality final was produced before this project was archived for the GitHub release.

This is a deliberately small, minimal Linear-style promo in the Infinite Payments aesthetic — a good exercise for students:

1. Lint the project: `npx hyperframes lint`
2. Run a standard render: `npx hyperframes render --quality standard --output renders/linear-promo-standard.mp4`
3. Compare your output to the draft `final.mp4` — the scenes are complete; you're upgrading the render profile.

If you spot scenes that need polish (timing, text stagger, color shifts), treat this as your own motion-graphics lab.
