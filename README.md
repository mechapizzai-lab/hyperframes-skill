## 🎯 C'est quoi ce projet ?

`hyperframes-skill` est un **skill HTML** qui te permet de générer des **hyperframes visuels** (compositions d'images, mockups, posts visuels) directement avec une IA. Parfait pour :

- 📱 Créer des **mockups** de posts réseaux sociaux
- 🎨 Composer des **visuels riches** sans Photoshop
- ⚡ Automatiser la **création de templates**
- 🚀 Itérer **vite** sur des concepts visuels

## 🚀 Démarrage rapide

```bash
# Clone le repo
git clone https://github.com/mechapizzai-lab/hyperframes-skill.git

# Suis les instructions du SKILL.md
cd hyperframes-skill
```

> 💡 **Ce skill est pensé pour être utilisé par un agent IA** qui va lire le `SKILL.md` et générer du HTML d'hyperframe à la demande.
